IS Workflow

This is a living document and is subject to change. Last updated: 6 June 2026
The Resources You Will Use

Open these first.

  • The internal Theme Guide found here.
  • This page in the IKB.
  • The submission tracker which can be found here.
  • The Duosuma submission manager accessed through Duotrope here.
Step 1

Access your Duotrope dashboard and select the oldest assigned submission.

Step 2

Download the Word document.

Step 3

Search for the submission using Ctrl+F and entering the submission ID. Open the file through the tracker in column Y.

Step 4

Move the downloaded Word document to the folder.

Step 5

Make sure the file opens in Google Docs and delete the file from your device.

Guideline Review

Our guidelines are not intended to act as a gate to publication, each guideline serves a purpose so please check them thoroughly.

  • Is the first page blank? If yes, move to the next guideline. If no, make a note in the Notes file and move on to the next guideline.
  • Does the word count fall within acceptable range (up to 6,000 words)? Word count can be viewed by using the tools menu or by pressing Ctrl+Shift+C. If yes, move to the next guideline. If no, make a note in the Notes file and move on to the next guideline.
  • Does the document have any Identifying information? Examples include:
    • Author Name
    • Author Address
    • Author phone number
    • Publication History
    • If you come across information that is possibly identifying, please escalate to the Senior Intake Specialist.
  • If a submission violates any of these guidelines you'll send a form rejection via Duosuma. Please make sure to notate the reasons for rejection in the Notes and select the appropriate letter based on the guidelines violated. Interns must obtain mentor approval before issuing a rejection.
Theme Assessment
  • Read through the entire story, notate any themes that the story might fit into.
  • Refer to the theme guide for current themes and the Issue Capacity graph on the traker dashboard.
  • In Duotrope, select all possible theme matches.
  • If there are no theme matches, or all possible themes are full, send appropriate rejection letter in Duosuma and document in Notes file. Interns must obtain mentor approval before issuing a rejection.
Update Tracker

Upon completion of these steps, update the tracker with your initials from the dropdown menu in column C and the Status to C in column D.

Wrap Up

Once everything is done, you will reassign the submission to the Senior Intake Editor assigned to your group and move on to the next submission.
